
# 📘 AI WAEC Math Question Generator

This project generates WAEC-standard mathematics questions with answers and explanations.

## 🚀 Features
- Algebra question generation
- Step-by-step explanations
- Easily extendable to other topics:
  - Differentiation
  - Integration
  - Coordinate Geometry
  - Financial Mathematics

## 🧠 Motivation
Built to support students preparing for WAEC exams and to explore AI-assisted education tools.

## 🛠 Tech Stack
- Python
- Random module (for now)
- Future: OpenAI API for intelligent question generation

## 📌 Future Improvements
- Add more topics (SSS1–SSS3)
- Difficulty scaling
- Web interface (Streamlit)
- AI-powered explanations
